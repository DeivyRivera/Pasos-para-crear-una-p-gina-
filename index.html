Pasos para crear una página en 000Webhost

1. **Registrarse en 000Webhost:

- Ve a [000Webhost](https://www.000webhost.com/). - Haz clic en "Sign Up for Free" o "Get Started for Free". - Llena el formulario con tu correo electrónico, contraseña y nombre del sitio web. También puedes registrarte usando una cuenta de Google o Facebook. - Confirma tu correo electrónico si te lo solicita.

2. **Crear un Nuevo Sitio Web:

- Una vez registrado y logueado, serás redirigido al panel de control. - Haz clic en "Create New Site". - Introduce un nombre para tu sitio web y elige una contraseña. Este será tu nombre de usuario para FTP y otros servicios.

3. **Elegir el Método de Construcción:

- Puedes optar por usar el "Website Builder" para una creación fácil y rápida con plantillas. - Alternativamente, puedes elegir "Upload Own Website" para subir tus propios archivos HTML, CSS, JavaScript, etc.
4. **Subir Archivos con el File Manager:
- Si seleccionaste "Upload Own Website", serás llevado al "File Manager". - Haz clic en "Upload Files Now". - Sube tu archivo `index.html` y cualquier otro archivo relacionado (imágenes, CSS, etc.).
5. **Uso de FTP (Opcional):
- También puedes usar un cliente FTP (como FileZilla) para subir tus archivos. - Encuentra la información de FTP en el panel de control bajo "Settings" > "General" > "FTP Details". - Conéctate usando la dirección del servidor, nombre de usuario y contraseña proporcionados. 6. **Publicar y Visualizar tu Sitio Web:** - Una vez que tus archivos están subidos, tu sitio web estará disponible en la URL proporcionada por 000Webhost (por ejemplo, `tusitio.000webhostapp.com`). - Puedes acceder a esta URL directamente desde tu navegador para ver tu sitio en línea.
